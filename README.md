### crime_statistics
Репозиторий для хранения датасета полученного для выполнения задания.

Описание полей:


| Поле | Описание |
| ------ | ------ |
| location | Место (Север, Юг) |
| town | Город (Liverpool, York и т.д.) |
| mortality | Смертность (целое число) |
| hardness | Жестокость (целое число) |
